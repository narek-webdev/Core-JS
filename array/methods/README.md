# Array Methods

This folder contains manual implementations of common JavaScript array methods.

The goal of these exercises is to better understand how native `Array.prototype` methods work by recreating their behavior with plain JavaScript.

## Implemented Methods

- [every.js](every.js)
- [filter.js](filter.js)
- [find.js](find.js)
- [findIndex.js](findIndex.js)
- [flat.js](flat.js)
- [forEach.js](forEach.js)
- [includes.js](includes.js)
- [map.js](map.js)
- [pop.js](pop.js)
- [push.js](push.js)
- [reduce.js](reduce.js)
- [reverse.js](reverse.js)
- [shift.js](shift.js)
- [slice.js](slice.js)
- [some.js](some.js)
- [unshift.js](unshift.js)

## Notes

Some implementations use custom method names such as `transform` and `accumulate` while following the behavior of native methods like `map` and `reduce`.

These files are meant for learning, experimentation, and practice.
